# Authoring

Builders over the generated models and the write path to a PDS. The
builders construct anchors, layers, and cross-references that validate
against the lexicons at construction time. The publish module diffs a
local Repository revision against a PDS and applies the minimal write
set. For the workflow see [Guides > Authoring and
publishing](../guide/authoring.md).
